Payroll Servicess WordPress Theme
================================

Installation
1. Copy the parolls-theme folder into wp-content/themes/.
2. Activate the theme from Appearance > Themes.
3. Set a site logo and customize theme options from Appearance > Customize.

Recommended plugins
- Contact Form 7 (optional, for advanced forms)
- Yoast SEO or Rank Math (optional, for SEO)
